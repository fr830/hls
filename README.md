SwordsDance.Hls
===============

Utilities for parsing HLS (HTTP Live Streaming) playlists.

(Very much a work in progress.)

Features
--------

* Read data from HLS playlists into typed tokens, complete with line and
  column information to facilitate debugging.

Dependencies
------------
